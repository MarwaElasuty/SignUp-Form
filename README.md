# How It Works

The user enters email and password.

Inputs are managed via state using useState.

Invalid inputs (missing @ in email or password < 6 characters) trigger conditional styling and error indicators.

Components update dynamically based on user interaction and validation state.

The project demonstrates how to combine multiple styling approaches and modular component structure in React.
